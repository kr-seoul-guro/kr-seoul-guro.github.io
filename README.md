# 구로구 · 서울특별시

> 구로구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-seoul-guro.github.io/**

구로구는 안양천을 사이에 두고 도심 제조업의 유산과 첨단 지식산업이 공존하는 서울 남서부의 관문형 자치구입니다. 1960년대 한국 최초의 수출산업공업단지였던 구로공단이 오늘날 서울디지털산업단지(G밸리)로 탈바꿈하면서, 봉제 재봉틀 소리 대신 IT·벤처 기업의 사무실이 들어선 산업 전환의 상징적 무대가 되었습니다. 구(區) 이름은 아홉 명의 장수한 노인이 살았다는 전설에서 유래한 '구로(九老)'로, 이는 오래된 마을의 역사성과 지역민의 자부심을 담고 있습니다. 대한민국 유일의 돔구장인 고척스카이돔, 폐선로를 되살린 항동철길, 서울 최초의 시립수목원인 푸른수목원 등 산업·문화·자연이 촘촘히 얽힌 볼거리를 품고 있습니다. 신도림역과 구로디지털단지역을 축으로 유동 인구가 집중되는 서남권 교통·상업의 결절점이기도 합니다.
